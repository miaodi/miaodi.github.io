# 1. Recursive

```c++
int numTrees(int n)
{
    if (n == 0 || n == 1)
        return 1;
    int res = 0;
    for (int i = 1; i <= n; i++)
    {
        res += numTrees(i - 1) * numTrees(n - i);
    }
    return res;
}
```

| Space | Time |
|-------|------|
|   $O(log(n))$    |   $O(c^n)$   |
